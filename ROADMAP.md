# EndTerraForged 旧路线说明

> 文档状态：已停止维护。
> 2026-07-11 起，长期路线统一由 [`GOAL.md`](GOAL.md) 维护，当前执行顺序统一由 [`PLAN.md`](PLAN.md) 维护。

旧版 ROADMAP 曾以 `min_y=-2032`、`height=4064` 作为默认目标，并混合记录了历史提交、阶段流水账和未来设想。该默认值已被 Standard `min_y=-256`、`height=512` 取代；4064 仅保留为未来显式选择且带警告的 Epic 规格。

为了避免后续开发者和模型误用历史方案，旧路线正文不再作为权威文档继续维护。需要追溯历史实现时请查看 Git 历史；制定和执行新任务时按以下顺序阅读：

1. [`AGENTS.md`](AGENTS.md)
2. [`GOAL.md`](GOAL.md)
3. [`MEMORY.md`](MEMORY.md)
4. [`PLAN.md`](PLAN.md)
5. [`docs/DOCUMENTATION_INDEX.md`](docs/DOCUMENTATION_INDEX.md)
